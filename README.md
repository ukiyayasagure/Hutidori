# ふちどり。

画像に「ふちどり（アウトライン）」を付与するための Windows 向けツールです。  
主に「伺か」用シェル画像を対象としています。

## これは何をするものか

主に「伺か」のシェルのような画像に対して、  
ふちどり処理を行うためのソフトウェアです。

以下のいずれかの条件に当てはまる画像を対象とします。

- 左上（0,0）の色を背景（透過）色として扱う画像  
- 32ビット画像（RGBA、αチャンネル付き）  
- 「伺か」の PNA ファイルを持つ画像  

---

## 動作環境

- **.NET Framework 3.5 SP1 以降** が必要です
- 比較的処理が重いため、ある程度性能のあるマシンでの実行を推奨します
